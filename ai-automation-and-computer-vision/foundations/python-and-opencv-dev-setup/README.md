# Python & OpenCV Dev Setup

### Learning Objectives

1. Create and activate a Python virtual environment using venv
2. Install OpenCV and verify the installation in a Python script
3. Open a webcam with cv2.VideoCapture() and read frames in a loop
4. Display a live frame in a window using cv2.imshow() and cv2.waitKey()
5. Overlay text on a frame using cv2.putText() with custom font, size, colour, and position
6. Calculate and display live frames-per-second (FPS) on the video feed
7. Gracefully release the webcam and close all windows on exit
