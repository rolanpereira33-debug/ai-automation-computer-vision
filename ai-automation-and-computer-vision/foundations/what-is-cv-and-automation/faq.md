# FAQ

<table data-header-hidden><thead><tr><th width="273.99993896484375" valign="top">Student question</th><th valign="top">Answer</th></tr></thead><tbody><tr><td valign="top">Does the computer actually understand what it sees?</td><td valign="top">No, it recognises patterns. It does not understand the way humans do. It has learned that certain pixel patterns usually mean 'ball'. That's pattern matching, not understanding.</td></tr><tr><td valign="top">How fast does it happen?</td><td valign="top">Modern systems run at 30–60 frames per second on a laptop. That means a decision every 17–33 milliseconds faster than a human blink.</td></tr><tr><td valign="top">Can it make mistakes?</td><td valign="top">Absolutely. CV systems fail on unusual angles, bad lighting, occlusion, or data they have never seen before. We will break our own models on purpose later to have fun.</td></tr><tr><td valign="top">Do we need the internet for this?</td><td valign="top">No. Everything we build runs locally on your laptop using your webcam. No cloud required though we will explore cloud APIs later.</td></tr><tr><td valign="top">Is this the same as AI?</td><td valign="top">CV is a branch of AI. AI is the broad field; machine learning is a technique within AI; CV is an application of machine learning to visual data.</td></tr></tbody></table>
